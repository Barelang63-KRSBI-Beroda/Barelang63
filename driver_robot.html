How to Install Library from Github

For clone repository, you can use this command:


        git clone https://github.com/bblanchon/ArduinoJson.git

To build and install the cloned library, run this:


        cd library_you_cloned
        mkdir build
        cmake ..
        sudo make install

Repeat this process for each cloned library.
