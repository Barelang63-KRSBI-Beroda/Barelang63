OpenCV with CUDA and cuDNN

Install OpenCV from this source with CUDA activation


        sudo rm -r /usr/include/opencv4/opencv2

1. Install dependencies


          sudo apt-get update
          sudo apt-get install -y libgtk2.0-dev pkg-config build-essential cmake git \
              libatlas-base-dev gfortran python3-dev \
              libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev \
              libavcodec-dev libavformat-dev libswscale-dev \

2. Clone repository OpenCV and OpenCV_contrib :


          git clone https://github.com/opencv/opencv.git
          git clone https://github.com/opencv/opencv_contrib.git

3. Make folder build


          cd opencv
          mkdir build
          cd build

4. Build OpenCV

Before that please check if you have CUDA_ARCH_BIN on your system, with this command :


          nvidia-smi --query-gpu=compute_cap --format=csv

input the value of CUDA_ARCH_BIN, based on the output of the command line above. Build opencv with Cuda activation with the command below: 


          cmake -D CMAKE_BUILD_TYPE=RELEASE \
                -D CMAKE_INSTALL_PREFIX=/usr/local \
                -D WITH_CUDA=ON \
                -D WITH_CUDNN=ON \
                -D OPENCV_DNN_CUDA=ON \
                -D OPENCV_EXTRA_MODULES_PATH=../../opencv_contrib/modules \
                -D INSTALL_PYTHON_EXAMPLES=OFF \
                -D INSTALL_C_EXAMPLES=OFF \
                -D WITH_GTK=ON \
                -D WITH_QT=OFF \
                -D BUILD_EXAMPLES=ON \
                -D CUDA_ARCH_BIN=8.6 \
                -D WITH_FFMPEG=ON \
                ..

5. If the build process is complete install opencv.


          make -j$(nproc)
          sudo make install
          sudo ldconfig

Making file opencv4.pc for OpenCV

1. Make file oepncv4.pc

Create a file named opencv4.pc in the /usr/local/lib/pkgconfig/ or /usr/lib/pkgconfig/ directory. Use a text editor like nano to create this file :


          sudo mkdir -p /usr/local/lib/pkgconfig/
          sudo nano /usr/local/lib/pkgconfig/opencv4.pc

2. Contents of opencv4.pc File

Fill in the opencv4.pc file as follows, adjust the library path (libdir) and header (includedir) if needed and fill in the opencv version based on the version that has been installed :


          prefix=/usr/local
          exec_prefix=${prefix}
          libdir=${exec_prefix}/lib
          includedir=${prefix}/include/opencv4

          Name: OpenCV
          Description: Open Source Computer Vision Library
          Version: 4.10.0
          Libs: -L${libdir} -lopencv_core -lopencv_imgproc -lopencv_highgui -lopencv_imgcodecs -lopencv_videoio
          Cflags: -I${includedir}

- prefix: OpenCV installation location on the system (usually /usr/local). - libdir: Path to the OpenCV library directory (/usr/local/lib). - includedir: Path to the OpenCV header (/usr/local/include/opencv4). - Libs: List of libraries to use. - Cflags: Path to header files required for code compilation. Be sure to customize the OpenCV version according to what is installed.

3. Set Variable PKG_CONFIG_PATH

Add the directory where the opencv4.pc file is located to the PKG_CONFIG_PATH variable, so that pkg-config can detect it :


          export PKG_CONFIG_PATH=$PKG_CONFIG_PATH:/usr/local/lib/pkgconfig/

To make this change permanent, add the above command to the .bashrc file :


          echo 'export PKG_CONFIG_PATH=$PKG_CONFIG_PATH:/usr/local/lib/pkgconfig/' >> ~/.bashrc
          source ~/.bashrc

4. Check Version with pkg-config

After the opencv4.pc file is created, check whether OpenCV has been detected by pkg-config with the following command: 


          pkg-config --modversion opencv4

If successful, the above command will display the OpenCV version, which is 4.10.0.
