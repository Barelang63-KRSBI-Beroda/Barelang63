Install STM32CubeIDE

Copy or download installer STM32CubeIDE, there is two source :

1. From barelang63 hdd with file name “en.st-stm32cubeide_1.8.0_11526_20211125_0815_amd64.deb_bundle.sh.zip”

2. From this linkGDrive

Do not use any version.

Extract STM32CubeIDE zip file

Then, inside the extracted folder, give access permission to the .sh file with the :


          chmod +x st-stm32cubeide_1.8.0_11526_20211125_0815_amd64.deb_bundle.sh/

Then, install it with the command :


          sudo ./st-stm32cubeide_1.8.0_11526_20211125_0815_amd64.deb_bundle.sh/

Then accept all licenses by typing y in the terminal and pressing enter.

to skip the license text press 'q'

then wait for the installation process to complete.

Open the STM32CubeIDE application in the application

1. Select the desired workspace location, then click “launch”

2. To install the desired board, create a new project by clicking “Start new STM32 project”. wait for the download process to complete.

3. Then select the board to be used, for example here we will use the “Nucleo-F767ZI” board. click “Board Selector”, then search for the board name in the “Commercial Part Number”, click the board, then click next.

4. Enter the desired project name, select the desired program language and click finish.

If you have not previously created an STM program with the selected board, the application will automatically download the required software packages.
