Build RTDB2

What is RTDB2 ?

Rtdb2 is the inter-robot communication package of MSL robocup.

This library from Team Cambada

Requirements

This project uses some 3rd-party libraries :

LZ4

zstd

LMDB

Msgpack

Xerces-c

xsdcxx

How to build


        git clone https://github.com/bblanchon/ArduinoJson.git

To build and install the cloned library, run this:


          - copy folder rtdb2 in hardisk barelang63
          - cd rtdb2
          - mkdir build
          - cd build
          - cmake -DCMAKE_BUILD_TYPE=Release ..
          - sudo make install

Install RTDB2 monitoring tools


          pip3 install lmdb msgpack-python
