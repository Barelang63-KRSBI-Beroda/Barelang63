Install Basestation

What is Basestation

Basestation is an application that is usually used to monitor/give commands to the robot during the game.

Requirements

There are several libraries required to create Basestation :

1. QT5 / QT6 2. VTK 9.* 3. Rtbd2

How to build VTK 9.

1. Copy the VTK 9.* folder from the Barelang 63 hard disk.

2. Extract the VTK 9.* tar file.

3. Enter the VTK 9.* directory: 


          mkdir build 
          cd build
          cmake .. -DVTK_GROUP_ENABLE_Qt=WANT -DVTK_QT_VERSION=5
          make -j$(nproc)
          sudo make install

How to build basestation

1. Copy the Basestation folder from the Barelang 63 hard disk.

2. Go to the KRSBI BERODA PROGRAM Folder and copy the basestation.tar file

3. Install in Terminal: 


          pip3 install nuitka

4. After copy the basestation.tar file, then extract the file to your directory and go to the basestation directory

5. Run the command below: 


          python3 -m nuitka basestation_build.py -o basestation_build
        

          python3 -m nuitka basestation_launch.py -o basestation_launch

6. after that move the basestation_launch and basestation_build executables to the /usr/bin/ folder with the command : 


          sudo mv basestation_build /usr/bin/
        

          sudo mv basestation_launch /usr/bin/

7. then we can build a basestation (Note 8 means how many cpu cores): 


          basestation_build 8

How to run Basestation

Run the Basestation application with the following command : 


          basestation_launch /build/basestation

To change the running parameters, you can use the following command : 


          basestation_launch build/basestation --config_file /opt/offline/config/cfg.xml --network_name agent2 --id 50
